# How to Commit Changes (Template)

## What is a commit?
- A commit is a saved snapshot of your changes

## Steps
- See changes:
  - `git status`
- Stage changes:
  - `git add .`
- Commit:
  - `git commit -m "Write a clear message"`

## Good commit messages
- Keep it short
- Say what you did
- Examples:
  - "Fix typo in README"
  - "Add practice task"
