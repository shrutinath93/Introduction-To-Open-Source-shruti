# How to Create a Branch (Template)

## Why branches?
- Branches keep your work separate from `main`
- Safer for beginners

## Steps
- Make sure you are in the repo folder
- Create a new branch:
  - `git checkout -b my-first-branch`

## Check your branch
- `git branch`
- The current branch has a `*`
