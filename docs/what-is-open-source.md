# What Is Open Source?

## Quick meaning
- **Open source** means the code is public
- People can:
  - Read it
  - Use it
  - Improve it
  - Share changes

## Why it exists
- Build software together
- Learn from real projects
- Fix problems faster

## Common words (simple)
- **Repository (repo):** a project folder on GitHub
- **Maintainer:** person who reviews and merges changes
- **Contributor:** person who helps by making changes
- **Issue:** a task or bug written on GitHub
- **Pull Request (PR):** your change request

## How contributing works
- You find a small task
- You change a file
- You open a PR
- A maintainer reviews it
- Your change gets merged

## Example contributions (great for beginners)
- Fix a typo in docs
- Improve instructions
- Add examples
- Add your name to a contributors list

## What you need to start
- A GitHub account
- Git installed
- Patience (everyone starts here)
