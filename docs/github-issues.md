# GitHub Issues (Template)

## What is an issue?
- A place to track:
  - Bugs
  - Tasks
  - Ideas
  - Questions

## How to pick an issue (beginner)
- Look for labels like:
  - `good first issue`
  - `beginner`
  - `documentation`

## How to write a good comment
- Keep it short
- Say what you want to do
- Example:
  - "Hi! I want to work on this. Can you assign it to me?"
