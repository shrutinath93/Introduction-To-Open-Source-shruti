# Practice Task: Simple Improvement

## Goal
- Make a useful improvement with a small PR

## Pick one
- Add a simple example command to any doc
- Improve one bullet point to be clearer
- Add a new FAQ question to the README

## Rules
- Keep it small (1 file is enough)
- Use simple words
- No big refactors

## PR title ideas
- "Add example to Git basics"
- "Clarify fork steps"

## Done?
- Open a PR and wait for review
