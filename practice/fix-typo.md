# Practice Task: Fix a Typo

## Goal
- Learn how to make a small, safe change

## What to do
- Find a tiny typo in any markdown file:
  - README
  - docs pages
  - practice pages
- Fix only the typo
- Do not change formatting unless needed

## Example
- Before: "thier"
- After: "their"

## Commit message ideas
- `Fix typo in docs`
- `Fix spelling in README`

## Done?
- Push your branch
- Open a PR
